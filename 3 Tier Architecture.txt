In our Daily Life:
1. we will use Android Apps/IoS Apps and Amazon, Facebook in Chrome browser.
2. When you use flipkart you will send a request(http). 
3. Load Balancer will take the request and sends to available/free web server.
4. web server will receive and forward to application server.
5. Application server will interact with database and prepares the data and sends back to web server.
6. Web Server will apply some decorations/tabular format/css and will show to the user on the screen.

Load Balancer/Web Server - Web Tier
Application Server - Application Tier
Database server - DB Tier

In between all the 3 tiers we have Network connections, routers and sub networks.
Who is Handling all these tiers: DevOps.
